# My 'sd' directory

Based on [sd](https://github.com/ianthehenry/sd).

```
❯ sd
sd commands

sysclean ... -- sysclean commands
```

```
❯ sd sysclean
sysclean commands

all    -- run all sysclean operations
brew   -- run `brew cleanup`
conan  -- clean conan builds from cache
docker -- run `docker system prune`
```
